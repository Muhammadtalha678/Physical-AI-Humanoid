# Accessibility Statement

We are committed to making the Physical AI & Humanoid Robotics textbook accessible to all learners, including those with disabilities.

## Our Commitment

This textbook has been designed with accessibility in mind, following the Web Content Accessibility Guidelines (WCAG) 2.1 AA standards.

## Accessibility Features

### Navigation
- **Keyboard navigation**: All content can be accessed using keyboard shortcuts
- **Logical tab order**: Navigation follows a logical sequence
- **Skip links**: Skip to main content links are available
- **Clear headings**: Proper heading hierarchy (H1, H2, H3) for screen readers

### Content
- **Alt text**: All images include descriptive alternative text
- **Color contrast**: Sufficient color contrast for readability
- **Text alternatives**: Non-text content has text alternatives
- **Captions**: Video content includes captions where applicable

### Interactive Elements
- **Focus indicators**: Clear visual indicators for focused elements
- **Descriptive labels**: Form elements have descriptive labels
- **Error prevention**: Clear error messages and prevention where possible

## How to Navigate

### Keyboard Shortcuts
- `Tab` / `Shift+Tab`: Navigate forward/backward through interactive elements
- `Enter` / `Space`: Activate links and buttons
- `Arrow keys`: Navigate within menus and interactive components

### Screen Reader Support
The site is compatible with popular screen readers including:
- NVDA
- JAWS
- VoiceOver (Mac/iOS)
- TalkBack (Android)

## Feedback

If you encounter accessibility barriers or have suggestions for improvement, please contact us at [accessibility-contact-email].

## Continuous Improvement

We continuously work to improve the accessibility of our content and welcome feedback from users to help us enhance the learning experience for everyone.
